# churn_predictions_machine-learning-and-data-science
# 🚀 Customer Churn Prediction using Machine Learning

## 📖 Overview

This project predicts whether a customer is likely to churn using Machine Learning algorithms. The dataset was preprocessed using data cleaning, label encoding, one-hot encoding, and feature scaling before training multiple classification models.

---

## 📂 Dataset

- IBM Telco Customer Churn Dataset

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## ⚙️ Machine Learning Workflow

- Data Collection
- Data Cleaning
- Handling Missing Values
- Label Encoding
- One-Hot Encoding (`pd.get_dummies`)
- Feature Scaling (`StandardScaler`)
- Train-Test Split
- Model Training
- Model Evaluation

---

## 🤖 Models Used

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

---

## 📊 Model Performance

| Model | Accuracy |
|--------|----------|
| Logistic Regression | **81.97%** |
| Decision Tree | **70.83%** |
| Random Forest | **78.99%** |

**Best Model:** Logistic Regression
